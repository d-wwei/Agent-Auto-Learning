# Install — Claude Code

## Auto Install

```bash
cd /path/to/auto-learning/mcp-server
npx auto-learning-mcp setup
```

## Manual Install

Append to `~/.claude/CLAUDE.md`:

```
@/path/to/auto-learning/cognitive-base/cognitive-protocol.md
```

Or copy the content of `cognitive-protocol.md` directly into CLAUDE.md.
